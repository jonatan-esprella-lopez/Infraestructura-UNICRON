# Código Procesal Civil de Bolivia — codigoProcesalCivil.pdf

## Resumen

Documento escaneado de **Código Procesal Civil de Bolivia**, compuesto por **152 páginas** en formato imagen (CCITT Fax / PDF escaneado). Contenido no extraíble como texto — requiere revisión visual directa.

Nombre de archivo recibido: `codigoProcesalCivil.pdf`.

## Aspectos a tener en cuenta

- **Formato**: PDF escaneado (image-only); no tiene capa de texto selectable. Cualquier análisis debe hacerse sobre el archivo visual o usando OCR.
- **152 páginas**: Volumen extenso — cubre todo el procedimiento civil ordinario y sumario en Bolivia.
- Típicamente contiene: jurisdicción, competencia, partes, demandas, pruebas, recursos, sentencias y ejecución de sentencias.
- Regula los **procedimientos para hacer valer** los derechos establecidos en el Código Civil (compraventa, arrendamiento, anticrésis, etc.).
- Específicamente relevante para casos de: incumplimiento contractual, resolución de ventas con reserva de propiedad, desalojo (arrendamiento) y restitución de bienes en anticrésis.

## Relación con otros archivos

- Complementa **codigoCivilBolivia.pdf**: mientras el Código Civil establece los derechos y obligaciones sustantivas, el Código Procesal Civil regula los pasos para exigirlos judicialmente.
- **Contrato_VentaConArras.doc** y **Contrato_ReservaPropiedades.doc**: En caso de resolución por incumplimiento de pagos, el procedimiento de ejecución o desalojo se rige por este código.
- **Contrato_ReservaPropiedades.doc**: La reserva de propiedad implica un trámite registral que debe cumplir el procedimiento establecido por este código.
- Todos los contratos hacen referencia a "órgano competente" o "autoridad llamada por ley" — sus definiciones responden a este código procesal.
