# Tools

Herramientas internas del monorepo. No contienen logica de negocio runtime.

npm run generate:module -- crm
npm run generate:api-module -- crm
npm run generate:web-module -- crm
npm run generate:component -- status-card
npm run generate:page -- crm leads
